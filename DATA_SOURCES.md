# Data Sources and Licensing

This project uses several external datasets that cannot be redistributed due to licensing restrictions. Follow the instructions below to obtain the required data files.

## Quick Setup

After cloning the repository, run:
```bash
python scripts/download_data.py
```

This will guide you through obtaining the required datasets.

---

## Required Data Sources

### 1. DrugBank Database

**License:** Academic License (free for non-commercial research) or Commercial License  
**Website:** https://go.drugbank.com/releases/latest

#### Files Required:
| File | Location | Description |
|------|----------|-------------|
| `full database.xml` | `data/` | Full DrugBank XML database |
| `drugbank.xsd` | `data/` | XML schema definition |

#### How to Obtain:
1. Create an account at [DrugBank](https://go.drugbank.com/)
2. Apply for an Academic License (requires institutional email)
3. Download "Full Database" in XML format
4. Place files in the `data/` directory

#### Licensing Notes:
- Academic license is free for non-commercial research
- Must cite DrugBank in publications
- Cannot redistribute the raw data

---

### 2. DDInter (Drug-Drug Interaction Database)

**License:** CC BY-NC-SA 4.0  
**Website:** http://ddinter.scbdd.com/  
**Citation:** Xiong et al., Nucleic Acids Research, 2022

#### Files Required:
| File | Location | Description |
|------|----------|-------------|
| `ddinter_all.csv` | `external_data/ddinter/` | All DDI records |

#### How to Obtain:
1. Visit [DDInter Download Page](http://ddinter.scbdd.com/download/)
2. Download the "All DDIs" file
3. Place in `external_data/ddinter/`

---

### 3. SIDER (Side Effect Resource)

**License:** CC BY-NC-SA 4.0  
**Website:** http://sideeffects.embl.de/  
**Citation:** Kuhn et al., Nucleic Acids Research, 2016

#### Files Required:
| File | Location | Description |
|------|----------|-------------|
| `drug_names.tsv` | `external_data/sider/` | Drug name mappings |
| `meddra_all_se.tsv.gz` | `external_data/sider/` | All side effects |
| `meddra_freq.tsv.gz` | `external_data/sider/` | Side effect frequencies |

#### How to Obtain:
1. Visit [SIDER Download Page](http://sideeffects.embl.de/download/)
2. Download the required files
3. Place in `external_data/sider/`

---

### 4. CTD (Comparative Toxicogenomics Database)

**License:** Custom (free for academic use)  
**Website:** https://ctdbase.org/  
**Citation:** Davis et al., Nucleic Acids Research, 2023

#### Files Required:
| File | Location | Description |
|------|----------|-------------|
| `CTD_chemicals_diseases.tsv.gz` | `external_data/ctd/` | Chemical-disease associations |

#### How to Obtain:
1. Visit [CTD Downloads](https://ctdbase.org/downloads/)
2. Download "Chemical-disease associations"
3. Place in `external_data/ctd/`

---

### 5. FAERS (FDA Adverse Event Reporting System)

**License:** Public Domain (US Government)  
**Website:** https://open.fda.gov/apis/drug/event/

#### Notes:
- FAERS data is accessed via API at runtime
- No pre-download required
- The file `external_data/faers_comprehensive_reports.json` contains cached queries and is included in the repository

## Directory Structure After Setup

```
data/
├── full database.xml         # DrugBank (obtain separately)
├── drugbank.xsd              # DrugBank schema
└── official_drug_class_references.md

external_data/
├── ddinter/
│   └── ddinter_all.csv       # DDInter (obtain separately)
├── sider/
│   ├── drug_names.tsv        # SIDER (obtain separately)
│   ├── meddra_all_se.tsv.gz
│   └── meddra_freq.tsv.gz
├── ctd/
│   └── CTD_chemicals_diseases.tsv.gz  # CTD (obtain separately)
├── faers_comprehensive_reports.json    # Included (public domain)
└── high_risk_drug_classes_reference.json  # Included

knowledge_graph_fact_based/   # Generated by build_fact_based_kg.py
├── knowledge_graph.pkl
├── statistics.json
└── neo4j_export/
    └── *.csv
```

## Building the Knowledge Graph

After obtaining all required data, build the knowledge graph:

```bash
python build_fact_based_kg.py
```

This generates:
- `knowledge_graph_fact_based/knowledge_graph.pkl`
- `knowledge_graph_fact_based/neo4j_export/*.csv`

## Troubleshooting

### "File not found" errors
Ensure all data files are placed in the correct directories as specified above.

### DrugBank license issues
- Academic licenses typically take 1-2 business days to approve
- Ensure you're using an institutional email address

### Large file handling
Some files are compressed (`.gz`). The code handles decompression automatically; do not manually extract them.
